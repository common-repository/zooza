This plugin enables integration of Zooza widgets to your website. Existing Zooza account is required for this plugin to work.

== Description ==

Simple integration of Zooza widgets for your website.

Supported widgets:

*   "Registration" used for registering new clients to courses
*   "Profle" displays user's profile
*   "Calendar" displays your courses and events in calendar form
*   "Video" displays purchased videos
*   "Checkout" provides checkout form for purchasing products

    The widgets will be shown below the main content of selected pages. Plugin will also synchronize URL's of selected widget pages back to Zooza.

== Installation ==

1. Upload `zooza.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup widgets in 'Main Settings ' > 'Zooza'
